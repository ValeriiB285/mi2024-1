1. **«Орієнтир-А»**
   Функціональність: зберігання подій у PostgreSQL, фільтрація за періодом/сектором/напрямком/типом, дашборд “швидкий огляд” з KPI та трендом, режим **executive** для оперативного розуміння загальної картини.

2. **«Сектор-Візор»**
   Функціональність: фокус на порівнянні секторів — heatmap “сектор × тиждень”, рейтинг секторів за інтенсивністю, drill-down до таблиці записів і деталей через FastAPI, режим **analyst** для глибокого аналізу.

3. **«Штаб-Інсайт»**
   Функціональність: автоматичні підсумки (короткий текстовий висновок) на основі KPI та трендів, готові “слайди-інсайти” у вигляді блоків дашборду, перемикач режимів **executive/analyst/demo** для різних аудиторій.

4. **«Оперативний Дашборд»**
   Функціональність: універсальна панель ситуаційних показників — KPI, тренд по днях/тижнях, 2 розподіли (за напрямками та типами), таблиця інцидентів з пагінацією і модальними деталями, централізована фільтрація.

5. **«Панорама-ОВУ»**
   Функціональність: “панорамний” огляд даних — агрегування по кількох зрізах одночасно, швидке перемикання періодів, порівняння “цей період vs попередній”, демонстраційний режим **demo/public** без чутливих полів.

6. **«Вектор-Аналіз»**
   Функціональність: виявлення основних напрямів активності — топ-напрямок, динаміка по напрямках, розподіли та рейтинги, фільтри з мінімальною інтенсивністю, вивід даних у форматах, зручних для аналітичної інтерпретації.

7. **«Рубіж-Монітор»**
   Функціональність: моніторинг змін — акцент на трендах і “сигналах” (стрибки інтенсивності, нетипові значення), швидкі зрізи по секторах/типах, таблиця подій для перевірки першоджерел (у межах навчальної моделі).

8. **«Компас-Дані»**
   Функціональність: навігація по даних — “від загального до конкретного”: KPI → тренд → розподіли → heatmap → таблиця → картка запису, зручний UX для демонстрації логіки аналітики різним аудиторіям.

9. **«Ситуаційний Огляд»**
   Функціональність: презентаційно-орієнтований дашборд — мінімум інтерфейсного “шуму”, чіткі підписи, короткі інтерпретації під графіками, режим **executive** як головний, плюс **demo/public** для відкритих показів.

10. **«Аналітик-Про»**
    Функціональність: навчальна “пісочниця” для аналітика — повний набір можливостей системи (БД+API+Web), розширені фільтри, сортування/пагінація, деталізація, відтворюваність через README та сценарій демо в репозиторії.
